# Customer Module

This module handles the end-user (customer) facing experience.

## Key Files
- `pages/Home.jsx`: Landing page and main browse view.
- `pages/Restaurants.jsx`: List of available restaurants.
- `pages/Menu.jsx`: Restaurant menu and item selection.
- `pages/Cart.jsx`: Shopping cart and checkout.

## Purpose
Allows customers to browse restaurants, view menus, add items to cart, and place orders.
