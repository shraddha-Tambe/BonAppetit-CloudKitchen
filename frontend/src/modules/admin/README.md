# Admin Module

This module contains the dashboard and management pages for the Platform Administrator.

## Key Files
- `pages/AdminDashboard.jsx`: Main entry point for the Admin dashboard. Handles internal routing and sidebar layout.
- `pages/DashboardOverview.jsx`: Overview statistics and charts.
- `pages/AdminSidebar.jsx`: Navigation sidebar component.

## Purpose
Allows admins to manage users, restaurants, delivery partners, and NGOs.
